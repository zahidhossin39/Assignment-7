# KeenKeeper

A dashboard app I built to help track and manage connections with friends. It basically acts as a personal CRM where you can see who you haven't talked to in a while, check contact goals, and log calls or text messages right from the app.

### Features

- **Friend Dashboard:** Inside Homepage There is an option for an "Add Friend" option and there you can add your friends and see all the friends you added.
- **Timeline Logging:**In the timeline section you will see all the interactions you had with your friends, like:
- giving them a call
- having a video call
- giving them a text message to them
- **Analytics Chart:**In the analysis stat section you'll see all the analytics for fiendship analytics and it's based on the timeline section, like how many calls you had, or texts you sent, or a video call you had with your friends. Those things you will show in analytics with a pie chart design.

### Technologies

- Next.js
- Tailwind CSS
- Recharts
- React-Toastify
